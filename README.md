# SpotifyApi

Objetivos: 

* Crear un servicio web que obtenga información desde la api de spotify
* Generar graficas utiles de la información 
* Crear una API para consulta de la información al servicio donde se entregue la información de la API de spotify con un procesamiento de los datos:
Por ejemplo si se consulta una fecha en spotify con formato dd/mm/yyyy se entrega por medio de nuestro servicio con otros formatos como dd-mm-yyyy


Fuentes de información de la API:
* https://developer.spotify.com/documentation/web-api
* https://developer.spotify.com/documentation/general/guides/authorization
